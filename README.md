# 🏦 Bank Management System (Flask)

A simple and modern **Bank Management System** built using **Python, Flask, HTML, CSS, JavaScript, and JSON**. This project demonstrates how a traditional command-line banking application can be transformed into a web application with a clean user interface.

## ✨ Features

* 👤 Create a new bank account
* 💰 Deposit money
* 💸 Withdraw money
* 📄 View account details
* ✏️ Update account information
* 🗑️ Delete an account
* 💾 Data stored locally in a JSON file
* 🌐 Responsive web interface

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/rajmurade/Bank-Management-flask.git
```

### 2. Move into the project folder

```bash
cd Bank-Management-flask
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python app.py
```

### 5. Open your browser

```
http://127.0.0.1:5000
```
